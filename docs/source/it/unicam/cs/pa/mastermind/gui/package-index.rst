it.unicam.cs.pa.mastermind.gui
==============================

Il seguente package contiene le classi relative a tutto ciò che concerne l'interfaccia grafica del progetto. Esso gestisce le varie interazioni con il menù di start e soprattutto le varie interzioni che i player compiono sul campo di gioco. Il ruolo di tutto ciò che è al suo interno è fondamentale per le meccaniche di gioco che vengono stabilite.
